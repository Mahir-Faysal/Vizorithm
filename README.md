# Another Algo Visualizer

Made this out of curiosity to explore my **`TypeScript`** understanding.
As for now I am working with `sorting` but I will gradually add more algorithms over time.

## Why this?

I tried to include features that other algorithm visualizers lack, such as:

- Pause during execution
- Adjust speed during visualization
- Check the steps at any point
